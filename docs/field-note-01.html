Field note 01

GitHub's 2FA notice was a useful forcing function.

The point is not drama. The point is legibility. If the account is active enough to receive the notice, the repository should be easy to inspect through code, approvals, operations, and task history.

Read the repo Read field notes Send a task outline

Current proof

What is already inspectable

  • desktop shell
  • local runtime API
  • multi-step operations in Pro
  • guarded terminal plus approvals
  • task transcript and support bundles

Scope

The claim is narrower than the ambition.

Klava is not being presented as a universal computer agent. The stronger claim is smaller: a local-first operations layer for machine work that needs notes, approvals, shell access, and a durable record.
